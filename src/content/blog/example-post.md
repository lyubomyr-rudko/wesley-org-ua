# Getting Started with Astro

Astro is a modern static site builder focused on performance and a great developer experience.

## Why Astro

Astro lets you build fast websites by shipping less JavaScript by default. That makes it a great choice for blogs, documentation sites, and content-focused projects.

## What makes it useful

- Fast page loads
- Great Markdown support
- Easy component integration
- Flexible content collections

## Example code block

```js
console.log("Hello from Astro!");
```

> Astro is a good choice when you want a clean, content-first site that stays fast and easy to maintain.
